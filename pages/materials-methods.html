Material and Methods

A number of steps were followed to extract, analyze and summarize the data. The following sub-sections describe the procedure in detail.

1) Data Extraction from Database

Tables containing information about reactions were downloaded from Mind The Gap Database using the relevant MySQL queries [18]. Scripts were written in Python [19] to extract and organize the name and number of species and reactions present within the database tables.

2) Sequence Retrieval and Organization

23S rRNA sequences for all the 141 organisms present in Mind The Gap Database along with the out-group organism (Thermoplasma acidophilum) were obtained from the NCBI Nucleotide Database. The sequences were then written to a single, FASTA format output file. The sequences were retrieved from NCBI using the Biopython module [20] of Python incorporated in Python script that produced the FASTA file.

3) Multiple Sequence Alignment

Multiple sequence alignment of the 23S rRNA sequences of 141 organisms plus one out-group organism was obtained through MAFFT 6.611 [21]. A threshold score of E=8.4e-11 was used.

4) Phylogeny Construction

Based on the results of multiple sequence alignment, a phylogeny was constructed using PhyML 2.4.4 [22]. PhyML uses a maximum likelihood approach to construct phylogenies. Bootstrappping was performed a 100 times on the tree to obtain the most likely phylogeny. For visualization of the phylogeny obtained and creation of phylogeny images, Dendroscope was used [23].

5) Ancestral State Reconstruction

For ancestral state reconstruction, Mesquite system was used [24]. Mesquite is a powerful, modular software to perform various analyses in evolutionary biology such as phylogenetic analyses and population genetics analyses. A character matrix was created for all the 2556 metabolic reactions using Python scripts. If a reaction was present in an organism, the number one was stored and if it a reaction was absent, a zero was stored.

Maximum Likelihood Ancestral State Reconstruction was performed for the categorical, discrete dataset of reactions. Likelihood reconstruction methods look for ancestor states that maximize the probability of producing the current state, under a defined model of evolution [25] [4].

Mesquite supports two models of evolution: the Markov k-state 1 parameter model (Mk1 model) and the Asymmetrical Markov k-state 2 parameter model (AsymmMK model). The AsymmMK model has two parameters. One parameter is for the rate of increase in state (from 0 to 1) and the other parameter is for decrease in state (from 1 to 0). For the given dataset, the AsymmMK model was, hence, used as it allowed to investigate the bias in gains versus loses [24].

In the ancestral state reconstruction, the out-group was removed from the phylogeny and the reaction traits for the out-group were not specified in the character matrix.

6) Metabolic Traits Phylogeny Construction

The construction of phylogeny based on metabolic traits was done through Pars programme in PHYLIP package [26]. Pars adopts a parsimony approach to construct phylogenetic trees, based on the Wagner parsimony method with multiple states. The Pars programme produced a total of 12 trees, from which a consensus tree was obtained using CONSENSE programme in PHYLIP package [26].

7) Analysis and Summarization of Results

All figures were created in R [27]. Additional data about the habitat and respiration types of organisms included in the study were provided by Dr. Pinney. δP values were calculated using scripts written in Python. The coloured phylogenies indicating the gain and loss of reactions based on δP values were made using the ape package in R. [27]
